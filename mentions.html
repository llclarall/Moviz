Présentation du Projet
Moviz est un projet universitaire collaboratif axé sur la data visualisation, permettant de découvrir des informations variées sur les films (popularité, genres, notes, etc.) via des graphiques interactifs. Basé sur la collecte de données via des API comme OMDb, le projet combine développement web et design UX/UI pour offrir une expérience fluide et intuitive.
