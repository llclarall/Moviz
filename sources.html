Sources

Cette section regroupe l’ensemble des documents et ressources utilisés dans le cadre du projet Moviz, incluant les outils, médias, et API exploités, ainsi que les inspirations et références méthodologiques.

1. Ressources Techniques et API

  • "Top 1000 Highest-Grossing Movies of All Time" : Liste IMDB utilisée pour récupérer les identifiants IMDb des 1000 films aux plus gros revenus. https://www.imdb.com/list/ls098063263
  • Wikidata Multi Search : Utilisé pour récupérer les ID AlloCiné, posters, et bandes-annonces des films. https://philippegambette.github.io/wikidataMultiSearch/
  • OMDb API : API gratuite avec options premium, utilisée pour la collecte de données sur les films. http://www.omdbapi.com/
  • Graphique sur les entrées au cinéma par année : Statistiques sur la fréquentation des salles en France de 2007 à 2023. https://fr.statista.com

  • Chiffres sur les sorties de films en France : https://www.cnc.fr
  • Chiffres sur les sorties de films en Amérique du Nord : https://www.statista.com
  • Chiffres sur tous les films sortis dans le monde : https://www.imdb.com
